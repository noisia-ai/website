# 02 — Product Vision

## La visión grande

Noisia puede convertirse en la interfaz para pedir, pagar, explorar y compartir inteligencia social.

Escenario de uso:

Un KAM de agencia recibe un request de campaña. No sabe todavía cómo abordarlo. Entra a Noisia, crea una cuenta, carga créditos, define una pregunta, configura mercados/plataformas/timeframe, corre un reporte y recibe un análisis de alto nivel con evidencia, dashboard narrativo y fuentes explorables.

Luego puede:

- Abrir la fuente original de cada insight.
- Ver tablas y quotes.
- Ajustar el reporte.
- Pedir otro ángulo al chat.
- Exportar como link privado.
- Descargar una PPT.
- Compartir un snapshot con cliente o equipo.

## Modelo mental

Noisia no es solo "social listening". Es una máquina de framing estratégico:

1. Recibe una pregunta de negocio.
2. Orquesta fuentes sociales y públicas.
3. Detecta patrones, tensiones, códigos culturales y actores de influencia.
4. Construye una narrativa estratégica.
5. Permite validar, editar, exportar y conversar con la evidencia.

## Superficies futuras

### Public Website

La puerta de entrada. Debe vender el universo Noisia y mostrar taste. No debe esconder el producto detrás de copy abstracto.

### Report Builder

Un flujo guiado para configurar investigaciones. Debe ayudar a formular mejores preguntas.

### Credit Wallet

Pay-per-event simple y transparente. El usuario entiende qué compra antes de gastar.

### Narrative Dashboard

El reporte como argumento. Primero conclusión, luego evidencia, luego exploración.

### Data Workspace

Zona densa para abrir fuentes, tablas, segmentos, quotes, filtros y vistas comparativas.

### Chat With Data

Una conversación con el reporte y sus fuentes. El chat debe citar evidencia, no inventar.

### Export Layer

Link compartible, PPT, snapshot, resumen ejecutivo y duplicación del reporte.

## Implicación para el website

Aunque el website sea público, debe insinuar estas capacidades. No hacer un "coming soon" pesado; mostrar escenas del producto, report cards, source drawers, prompt chips, credit wallet y export bar como señales del futuro.
